Nombre de referencia

RAL — Diseño de Red de Adaptación tipo L.

Calling Sequence

[Xp,Xs] = RAL(Rg,Rl)

Parameters

Rg

Resistencia del Generador

Rl

Resistencia de la carga

Xp

Reactancia Paralelo de la red de Adaptación

Xs

Reactancia Serie de la red de Adaptación

Description

Calcula las impedancias necesarias para adaptar dos impedancias, usando una red L

Examples

        [x,y]=RAL(100,75)
