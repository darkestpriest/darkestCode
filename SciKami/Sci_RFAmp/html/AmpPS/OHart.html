Nombre de referencia

OHart — Diseño de un oscilador Hartley.

Calling Sequence

[c1,c2,l3] = OHart(gm,f,rin)

Parameters

gm

Transconductancia del Transistor

f

Frecuencia del Oscilador a diseñar

rin

Impedancia de entrada de entrada

l1

L1 del Oscilador Hartley

l2

L2 del oscilador Hartley

C3

L3 del Oscilador Hartley

Description

Esta función realiza el cálculo de los componentes necesarios para la implementación de un Oscilador Hartley

Examples

        [x,y,z]=OHart(0.02,100e6,500)
