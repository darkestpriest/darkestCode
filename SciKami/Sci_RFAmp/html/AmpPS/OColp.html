Nombre de referencia

OColp — Diseño de un oscilador Colpitts.

Calling Sequence

[c1,c2,l3] = OColp(gm,f,rin)

Parameters

gm

Transconductancia del Transistor

f

Frecuencia del Oscilador a diseñar

rin

Impedancia de entrada de entrada

c1

C1 del Oscilador Colpitts

c2

C2 del oscilador Colpitts

l3

L3 del Oscilador Colpitts

Description

Esta función realiza el cálculo de los componentes necesarios para la implementación de un Oscilador Colpitts

Examples

        [x,y,z]=OColp(0.2,100e6,500)
