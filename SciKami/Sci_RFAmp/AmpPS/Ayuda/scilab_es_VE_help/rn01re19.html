RVirtPi — Cálculo de la R visrtual de una red Pi.

Calling Sequence

R = RVirtPi(Rh,Q)

Parameters

Rh

Resistencia de mayor magnitud

Q

Factor Q del circuito

R

R virtual de la red Pi

Description

Esta función calcula la R Virtual de una Red Pi

Examples

        R=RVirtPi(100,16)
